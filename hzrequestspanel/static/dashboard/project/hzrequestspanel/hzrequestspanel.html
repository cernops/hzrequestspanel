Update quota for "Cloud Infra Services"

Cloud project owners who want to change the resource quota for this project should click "Request quota change" button and fill the form.

Submiting the form, you will create a new SNOW ticket for your request. From that point, your request will be manage by Cloud Team from SNOW application. After submitting your ticket, you will receive a confirmation email.

For details on Cloud projects, please read http://clouddocs.web.cern.ch/clouddocs/details/quotas.html
